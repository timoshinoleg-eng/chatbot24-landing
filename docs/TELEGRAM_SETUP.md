# Настройка Telegram Bot

## 1. Создать бота
- Напишите @BotFather
- Создайте нового бота: `/newbot`
- Скопируйте токен в `TELEGRAM_BOT_TOKEN`

## 2. Добавить бота в канал
- Добавьте бота администратором в канал @ml_digest_daily
- Дайте права: "Чтение сообщений"

## 3. Получить ID канала
- ID канала: `3587382769` (уже указан в коде)
- Или используйте @userinfobot

## 4. Проверка
- Отправьте тестовое сообщение в канал
- Запустите синхронизацию вручную
